# Calendar App Usage Guide

## Version Information
- Version: v3.0
- Updated: April 16, 2026

## 1. Product Overview

This is a clean and efficient native Android calendar app focused on helping users manage their time and schedules effectively. Version 3.0 adds weather and alarm features, providing users with an all-in-one schedule management experience.

### Key Features
- ✅ **Multi-view Calendar**: Support for Month/Week/Day/Year views
- ✅ **Smart Reminders**: Flexible reminder settings, never miss important events
- ✅ **Weather Functionality**: Weather information displayed on calendar, support for multiple cities
- ✅ **Alarm Functionality**: Flexible alarm settings with snooze and statistics
- ✅ **Data Security**: Local storage, safe and reliable

---

## 2. Core Features

### 1. Calendar Views

#### Month View
- View complete monthly calendar layout
- Events marked on dates
- Weather icon and temperature shown in top-right corner
- Click date to view day details

#### Week View
- View detailed weekly schedule
- Current weather information displayed at top
- Weather shown for each time period
- Swipe left/right to change weeks

#### Day View
- View detailed daily schedule
- Events displayed on timeline
- Current weather shown at top

#### Year View
- Quick browse through entire year
- Click month to jump to month view

### 2. Event Management

#### Creating Events
1. Tap the "+" button in bottom-right corner of home page
2. Enter event title
3. Select start and end times
4. Set location (optional)
5. Add reminder: 5/15/30 minutes before, 1 hour, 1 day, or custom
6. Set repeat rule (daily, weekly, monthly, yearly, or custom)
7. Select calendar category (work, personal, family, etc.)
8. Add tags (optional)
9. Tap "Save"

#### Viewing and Editing Events
- Tap event in event list to view details
- Tap "Edit" on detail page to modify event
- Tap "Delete" to remove event

### 3. Weather Functionality

#### Viewing Weather
1. View weather at top of home page or in calendar cells
2. Tap weather area to enter weather detail page

#### Weather Detail Page Features
- **Current Weather**: Displays temperature, weather icon, feels-like temp, humidity, wind speed, wind direction, pressure, visibility
- **7-Day Forecast**: Weather and temperature for next 7 days
- **24-Hour Forecast**: Hourly weather details
- **Life Index**: Clothing index, sports index, UV index, car wash index, cold index, travel index
- **Air Quality**: AQI index, air quality level

#### City Management
- **Add Current Location Weather**: One-tap to get current location weather (requires GPS permission)
- **Add City**: Manually search and add other cities
- **Switch Cities**: Drop-down select at top of weather detail page
- **Delete City**: Remove unwanted cities in city management
- **Set Default City**: Set preferred city

### 4. Alarm Functionality

#### Alarm List
- View all alarms
- Shows time, repeat rule, label
- Quick enable/disable with switch on right

#### Creating Alarms
1. Tap "+" button in bottom-right corner of alarm page
2. Select alarm hour and minute
3. Set repeat rule (once, daily, weekdays, weekends, or custom days)
4. Enter label (e.g., Wake up, Work, Exercise, etc.)
5. Select ringtone
6. Enable/disable vibration
7. Set snooze: snooze interval (5/10/15/30 minutes), snooze count (1/3/5/unlimited)
8. Tap "Save"

#### Alarm Statistics
- View ring count
- View snooze count
- View average wake-up time

### 5. Search Functionality
- Enter keywords to search event titles, locations, descriptions
- Filter by date range, calendar category, tags
- Tap search results to view details

### 6. Settings

#### Appearance Settings
- Switch between dark/light mode
- Select theme color

#### Reminder Settings
- Set default reminder time
- Select reminder ringtone
- Enable/disable vibration

#### Calendar Management
- Create/edit/delete calendar categories
- Set calendar colors

#### Weather Settings
- Temperature unit: ℃/℉
- Auto-update frequency
- Display settings
- Default city

#### Alarm Settings
- Default ringtone
- Default vibration
- Default snooze settings

## 3. Quick Start

### First Time Use
1. Open app, month view displayed by default
2. Tap any date to view events for that day
3. Tap "+" button to create first event

### Adding Weather
1. Tap weather area on home page or go to weather page
2. Tap "Add Current Location Weather" (recommended) or manually search for city
3. Allow location permission (if using auto-location)
4. Start enjoying weather service

### Setting Alarms
1. Go to alarm page
2. Tap "+" button
3. Set time and repeat rule
4. Save alarm

## 4. FAQ

### Q: What if weather data doesn't update?
A: Please check your network connection, or tap "Refresh" button on weather detail page to manually update.

### Q: How to import/export calendar data?
A: Go to Settings → Data Management, select import or export (supports ICS format).

### Q: What if alarm doesn't ring?
A: Please check:
1. If alarm switch is on
2. If device volume is sufficient
3. If app has notification permissions
4. If device is in Do Not Disturb mode

### Q: How to switch views?
A: Tap view switcher at top of home page (Month/Week/Day/Year).
